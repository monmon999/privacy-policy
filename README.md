# プライバシーポリシー

Monmon（以下、「当方」といいます）は、スマートフォン向けゲームアプリ「チェインクエスト」（以下、「本アプリ」といいます）における、ユーザーの情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

## 1. 収集する情報およびその利用目的
本アプリでは、広告配信および利用状況分析のため、以下の第三者配信のツールを使用しています。これらのツールは自動的にデータを収集する場合があります。収集される情報、利用目的、第三者への提供等については、各ツールのプライバシーポリシーをご確認ください。
* Google AdMob（Google Inc.）https://policies.google.com/privacy 広告配信のため，広告識別子，デバイス情報，利用状況等を収集
* Apple GameCenter https://www.apple.com/legal/privacy/ ランキング機能の提供のため，スコア等をAppleが処理

本アプリはこれらの第三者配信ツール以外で一切情報を収集致しません．

## 2. プライバシーポリシーの変更
当方は、法令の変更等に伴い、本プライバシーポリシーを予告なく変更することがあります。
